Über MySaskia

MySaskia ist ein lokales Lernprojekt und gleichzeitig der Anfang eines kleinen Tier-Marktplatzes. Nutzer können Tiere entdecken, eigene Inserate erstellen und ihre Angebote über ein persönliches Konto verwalten.

Marktplatz ansehen Tier inserieren Mein Konto

Die Idee

🐾

Ein Tier-Marktplatz

MySaskia soll eine Plattform werden, auf der Nutzer Tiere kaufen, verkaufen und später auch Zubehör wie Futter oder Spielzeug anbieten können.

📸

Inserate mit Bildern

Verkäufer können Tier-Inserate mit Titel, Beschreibung, Tierart, Rasse, Alter, Preis und Bild erstellen.

👤

Eigener Konto-Bereich

Jeder eingeloggte Nutzer kann sein Profil bearbeiten, eigene Inserate verwalten und Support-Tickets erstellen.

Warum dieses Projekt?

🧠

Lernen durch Bauen

Das Projekt hilft dabei, wichtige Grundlagen moderner Webentwicklung zu verstehen: Frontend, Backend, REST-API, Datenbank, Login, Admin-Rechte und Datei-Uploads.

🖥️

Lokale Entwicklungsumgebung

MySaskia läuft lokal in einer Ubuntu-Desktop-VM. Das Frontend wird über Apache ausgeliefert, das Backend läuft als Flask-API mit SQLite-Datenbank.

🚀

Schrittweise Erweiterung

Aktuell entsteht die erste Version mit Tier-Inseraten. Später können Suche, Käufe, Favoriten, Nachrichten, Bewertungen und weitere Produktarten ergänzt werden.

Technik im Hintergrund

🌐

Frontend

Die Oberfläche besteht aus HTML, CSS und JavaScript. Sie nutzt eine gemeinsame Shop-Navigation, Kartenlayouts, Dark Mode und lokale Einstellungen.

⚙️

Backend

Das Backend basiert auf Python und Flask. Es stellt REST-Endpunkte für Login, Profile, Support-Tickets, Accounts und Tier-Inserate bereit.

🗄️

Datenbank

Die Daten werden aktuell in SQLite gespeichert. Dazu gehören Nutzer, Sessions, Tickets und Marktplatz-Inserate.
